In this guide, you will build a sentiment classifier, run it against a product review, and then optimize it automatically using a small set of labeled examples — all without writing a single prompt by hand. By the end, you will have a saved, optimized program you can reload and reuse in production.

## Prerequisites

Before you begin, make sure you have the following ready:

- **Python 3.10 or higher** installed on your machine
- **pip** available in your terminal
- An **OpenAI API key** (or a key for another supported provider — see [Installation](/installation) for alternatives)
